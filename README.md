### Build

Run tests, lint and create babelified files into /dist folder.

```
npm run build
```

### Watch

Watch source files and rebuilds on any change.

```
npm run watch
```

### Lint

Run eslint on /src folder

```
npm run lint
```

### Tests

Run tests on /src folder with Jest

```
npm run watch
```

### Coverage

Get tests coverage

```
npm run coverage
```
